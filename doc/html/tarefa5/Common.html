type Map = [String]

Tipo que representa o mapa do jogo.

type Pos = (Int, Int)

Tipo que representa uma posição.

isRock :: Pos -> Map -> Bool

Função que verifica se existe uma pedra numa dada posição.

isBrick :: Pos -> Map -> Bool

Função que verifica se existe um tijolo numa dada posição.

isInPos :: Pos -> Map -> [Char] -> Bool

Função que verifica se existem algum Powerup, bomba, ou jogador (dependendo do terceiro argumento) numa dada posição.

>>> isInPos (3,8) ["! 3 8"] ['!'] = True

>>> isInPos (3,8) ["2 3 8"] ['0','1','2','3'] = True

>>> isInPos (3,8) ["+ 3 8"] ['*'] = False

>>> isInPos (5,2) ["+ 3 8"] ['+'] = False

reactToEdge :: Map -> Pos -> (Int, Int) -> Int -> ((Int, Int), Int)

Função que muda a direção de um vetor quando a dada posição de aproxima um dado número de blocos da borda, e aumenta este último número sempre que muda de direção para cima. A funçäo determina em que direção a espiral deverá se mover no próximo tick.
